 Лабораторна робота №2 — Smart Timer (Qt, C++)

Цей проєкт є графічним застосунком, розробленим мовою C++ з використанням фреймворку Qt 6.
Його мета — реалізувати систему «розумного таймера», яка дозволяє створювати, редагувати, запускати, зупиняти та зберігати кілька таймерів одночасно.
Кожен таймер працює в реальному часі, автоматично оновлюється та виводить сповіщення після завершення відліку.

⚙️ Основний функціонал

 Додавання нових таймерів із заданою назвою та тривалістю

 Запуск і зупинка кожного таймера окремо

 Автоматичне збереження активних таймерів у файл data/timers.json

 Відновлення стану під час наступного запуску

 Сповіщення після завершення таймера

 Використані технології

Мова: C++

Фреймворк: Qt 6

Система збірки: CMake

Середовище: CLion / Qt Creator

 Основні класи

Timer — описує окремий таймер, зберігає назву, тривалість, стан; методи start(), stop(), tick()

TimerManager — керує колекцією таймерів, викликає tick() щосекунди, зберігає/завантажує з файлу

MainWindow — головне вікно програми з таблицею таймерів

AddEditDialog — діалогове вікно для створення або редагування таймера

 Запуск програми

Відкрити проєкт у CLion або Qt Creator

Зібрати через CMake
